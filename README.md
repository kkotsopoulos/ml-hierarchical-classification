# Machine Learning Pipeline: Hierarchical Classification 

A complete machine learning workflow developed in Python to solve a complex multiclass classification problem. This project implements an end-to-end predictive modeling pipeline, from raw data preprocessing to the deployment of a custom hierarchical classification architecture.

##  Project Overview & Features

* **Statistical Analysis & MLE:** Estimation of Gaussian distribution parameters (Maximum Likelihood Estimation) for dataset classes.
* **Feature Engineering:** Implemented dimensionality reduction using **PCA** (Principal Component Analysis) and data scaling strategies (StandardScaler, Normalizer).
* **Hierarchical Classification:** Designed a custom multi-stage hierarchical classification logic to improve predictive performance on heavily overlapping classes.
* **Model Training & Evaluation:** Utilized **Support Vector Machines (SVM)** with hyperparameter tuning, validated through K-Fold Cross-Validation.

##  Tech Stack & Libraries

* **Language:** Python 3
* **Machine Learning:** Scikit-Learn
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn

##  Repository Structure

* `Part_AC_Gaussian_Estimation.ipynb` - Jupyter Notebook covering data visualization and Maximum Likelihood Estimation of Gaussian parameters.
* `Part_D_Hierarchical_Classification.ipynb` - The core ML pipeline featuring data preprocessing, PCA, and the custom hierarchical SVM classifier.
* `ML_Project_Report.pdf` - Comprehensive technical documentation, confusion matrix analysis, and architectural decisions.
* `predicted_labels.npy` - Final exported predictions generated by the hierarchical model on the unlabeled test set.

##  How to Run

1. Clone the repository and ensure you have Python 3.x installed.
2. Install the required dependencies: `pip install numpy pandas scikit-learn matplotlib seaborn`
3. Run the Jupyter Notebooks sequentially to reproduce the data analysis and train the hierarchical classifier.
